# Rise — UI Mockup Image Prompts

These prompts generate clean, realistic UI mockup screenshots for the Rise tournament bracket platform. Each image should look like a real product screenshot — not an illustration or cartoon.

## Visual Design Language for All Mockups

- Dark UI theme: deep navy background (#0a0e1a), with lighter card surfaces (#1a1f35)
- Electric lime (#c8ff00) for active states, winners, CTAs, and highlights
- Hot coral (#ff4d6a) for eliminations and alerts
- White text on dark backgrounds, with muted gray (#5a6178) for secondary text
- Font style: clean geometric sans-serif (like Outfit), bold weights for headings
- Rounded corners on all cards and containers (12-16px radius)
- Subtle glow effects behind active/winning elements
- The Rise logo ("RISE" in uppercase lime text) appears in the top-left nav bar
