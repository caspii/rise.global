How a {{ n }}-Team Round Robin Works

In a round robin, every team plays every other team — no one is eliminated, and the final standings reflect performance across the whole event rather than a single knockout result. With {{ n }} teams that means {{ matches }} games spread over {{ rounds }} rounds, and each team plays {{ n_minus_1 }} matches.{% if odd == 1 %} Because {{ n }} is an odd number, one team sits out (a "bye") in each round — there's no way around it, so build the bye into your schedule and rotate it fairly, which the fixture above already does.{% else %} With an even number of teams, every team is in action every round, so all {{ games_per_round }} games can run in parallel if you have the courts or fields.{% endif %}

Round robin is the fairest format and the one rec leagues, clubs, and group stages rely on, because a single bad game doesn't send anyone home. The trade-off is game count — round robins grow quickly as you add teams — so for larger fields many organizers split into two pools, run a round robin within each, then send the top finishers to a knockout bracket. Standings are ranked by wins first, then point differential as the tiebreak, then head-to-head.

The painful part of running one by hand is keeping the standings current as scores come in, and telling everyone which game is next.
